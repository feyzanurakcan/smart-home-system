# Smart Home System

This project is a Java-based smart home system simulation.

## Features
- Light, Thermostat, and Alarm devices
- Object-oriented design
- Observer pattern for fault detection
- Console-based simulation

## Technologies
- Java
- Object-Oriented Programming (OOP)

## What I Learned
- Designing classes and system structure
- Using inheritance and polymorphism
- Applying the Observer pattern in a simple project
